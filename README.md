# Predicting Customer Churn for a Subscription Streaming Service

## Overview

This project predicts customers likely to cancel their subscription using Machine Learning techniques.

The objective is to help subscription businesses identify high-risk customers and improve customer retention strategies.


## Dataset

Dataset:
Telco Customer Churn Dataset

Features include:
- Customer demographics
- Subscription details
- Billing information

Target:
- Churn (Yes/No)


## Machine Learning Techniques

- Exploratory Data Analysis
- Feature Selection
- Cross Validation
- Hyperparameter Tuning
- Multi Model Comparison


## Models

Implemented:

- Decision Tree Classifier
- Random Forest Classifier
- Support Vector Machine


## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Confusion Matrix
